# Shark JS

## Demo

[Watch the demo video](fullscreen.mp4)

A fun and interactive web animation featuring a shark that follows your mouse cursor around the screen.

## Description

This project creates an animated shark using SVG graphics that smoothly follows the mouse cursor as you move it across the web page. The shark is composed of multiple layered parts that create a realistic swimming motion effect.

## Features

- Smooth mouse-following animation
- SVG-based vector graphics for crisp rendering at any size
- Responsive design that works on different screen sizes
- Lightweight and performant JavaScript implementation

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES5)
- SVG (Scalable Vector Graphics)

## How to Run

1. Clone or download this repository
2. Open `index.html` in any modern web browser
3. Move your mouse around the page to see the shark follow!

## Browser Support

Works in all modern browsers that support SVG and JavaScript:
- Chrome
- Firefox
- Safari
- Edge

## Project Structure

- `index.html` - Main HTML file
- `style.css` - CSS styles for layout
- `index.js` - JavaScript code for animation logic
